🎯Guess My Number!
A fun number-guessing game built with vanilla JavaScript as part of Jonas Schmedtmann's "The Complete JavaScript Course 2025: From Zero to Expert!"
🕹️ How to Play
The game picks a secret number between 1 and 20. Your job is to guess it!

Enter your guess and click Check
You'll get a hint: the number is too high, too low, or... you nailed it! 🎉
Each wrong guess costs you a point — you start with 20
Try to beat your highscore by guessing with as few attempts as possible
Click Again to reset the game and try once more

🛠️ Built With

HTML5
CSS3
JavaScript (DOM manipulation, event listeners, game logic)
